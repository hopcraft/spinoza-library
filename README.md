# Spinoza Library
> Implementation of some classical programming concepts, such as promise, async/await, mvvm, vdom, csp, etc.


## Running steps
1. Install Dependencies
```shell
$ npm i
```

2. Developement
```shell
$ npm start
```

3. Test
```shell
$ npm test
```

4. Build
```shell
$ npm build
```

5. Publish
```shell
$ npm publish
```
